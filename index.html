Cómo configurar
  1. Andá a script.google.com → Proyecto nuevo
  2. Borrá el código de ejemplo y pegá el script de abajo
  3. Guardá con Ctrl+S
  4. Click "Implementar" → "Nueva implementación"
  5. Tipo: App web · Acceso: Cualquier persona
  6. Copiá la URL y pegala arriba. ¡Listo! 🎉
